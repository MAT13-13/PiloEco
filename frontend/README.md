# 🐦 PiloEco

> **Le copilote financier intelligent qui aide chacun à économiser de l'argent simplement, sans changer ses habitudes.**

---

## 🚀 À propos

PiloEco est une plateforme SaaS qui analyse les dépenses du quotidien afin d'identifier automatiquement les économies possibles.

Contrairement à un simple comparateur de prix, PiloEco agit comme un véritable copilote financier.

Il accompagne l'utilisateur dans le temps :

- 📊 Analyse ses dépenses
- 🎯 Priorise les meilleures économies
- 🐦 Explique les recommandations avec Pilo
- 📈 Suit sa progression
- 🏆 Récompense les objectifs atteints

L'objectif est de permettre à chacun d'économiser plusieurs centaines d'euros par an en quelques minutes.

# 🚀 Installation

```bash
git clone https://github.com/MAT13-13/PiloEco.git

cd PiloEco/frontend

npm install

npm run dev
```

Application disponible sur :

```
http://localhost:3000
```
