# 📌 Règles du projet

Chaque version doit respecter les étapes suivantes :

- Fonctionnalité terminée
- Tests effectués
- Documentation mise à jour
- Commit Git
- Push GitHub

Aucune version n'est considérée comme terminée sans ces cinq étapes.
